# Agent Wiki

This package provides the canonical agent-wiki skill set for working inside the current project only. It is meant to be installed into one project at a time, never globally.

The canonical set is organized around six skills:

- `using-agent-wiki` starts you on the right workflow.
- `agent-wiki-bootstrap` creates a new wiki in the current project.
- `agent-wiki-structure` defines page types, links, and frontmatter.
- `agent-wiki-ingest` turns raw sources into stable wiki pages with traceability.
- `agent-wiki-query` answers from the wiki and writes back new stable knowledge when appropriate.
- `agent-wiki-lint` checks graph and content health, including orphan pages, broken links, duplicates, and stale content.

Start with `using-agent-wiki`, then use the focused skills as needed.
